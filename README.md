# Digital Design & VLSI Components Library

A comprehensive library of parameterized digital hardware components, ranging from basic combinatorial logic to complex arithmetic units.

## Project Structure
- **/rtl**: Verilog/SystemVerilog source files.
- **/verification**: Testbenches and formal verification scripts.
- **/constraints**: SDC files for timing and area constraints.
- **/docs**: Design specifications and timing reports.

## Toolchain
- **Logic Synthesis**: Yosys / OpenROAD
- **Environment**: Nix / Docker
- **Physical Design**: OpenLane2 (Nangate 45nm / ASAP7 7nm)
